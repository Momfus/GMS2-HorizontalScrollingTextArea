			- Cada texto tiene una bandera de "duplicado"
			- Esta bandera indica si el texto se esta pasando mas de una vez
			- Una vez que el texto termina de entrar a la pasarela, si busca para agregar otro texto y no hay: 
						-> Toma el que esta en la cabeza de la lista en movimiento

								--> Busca el siguiente que haya en la lista y repete el proceso
